## OPSWAT Metadefender Kiosk Toolkit

### To run in devlopment

```

1.git clone repository 

2. cd KioskToolKit 

3. $ npm i 

4. $npm run dev 

# with npx

$ npx create-nextron-app my-app --example with-material-ui


```

### To run in production 

```

1.git clone repository 

2. cd KioskToolKit 

3. $ npm i 

4. $npm run build:win

5. open .exe file

# with npx

$ npx create-nextron-app my-app --example with-material-ui


```
